Configuring the AI Moderator in Userology

General → Study Settings | Updated: 2025-03-17

AI configuration allows you to customize the AI moderator's language, voice, avatar, speaking, listening and silence settings, to perfect it as you like.

Accessing the AI Configuration Feature

To access the AI configuration feature, navigate to the Study settings from left nav bar. Here, you will find the AI configuration tab.

 

 

Modifying Avatar Library and Voice Settings

In the AI configuration tab, you can modify the avatar library and adjust various voice settings such as age, gender, and accents.

 

Understanding the AI Configuration Tabs

The AI configuration feature comprises three tabs: Speaking, Listening, and Silence. Each tab controls different aspects of the AI moderator's behavior.

Speaking Tab

The Speaking tab includes settings for:

  1. Speech normalization - Setting for speaking out the number out loud or not. For example - 22 will be read as "twenty two" with this setting enabled.

  2. Responsiveness - For controlling how responsive the AI moderator. Smaller numbers mean longer wait times for AI to speak after the participants response and vice versa.

  3. Speaking speed - For controlling the speed with which the AI moderator speaks.

  4. Interruption sensitivity - Controls how easily the AI moderator will be interrupted by the participants responses.

  5. Voice emotional intensity - Controls the tonality in the voice of the AI moderator.

  6. The AI's first message - You can set an opening message for starting the interview with.

Listening Tab

The Listening tab allows you to:

  1. Select input languages

  2. Specify expected keywords

These keywords help the AI moderator recognize new terminologies or proper nouns used in your study.

Silence Tab

The Silence tab includes settings for:

  1. Think out loud nudges - You can set the duration after which the AI moderator will prompt participants to speak if they remain silent for a specified period.

  2. End call settings - Set a duration for ending the call after a specified time limit.

  3. Active listening - Use responses like "yeah", "hmm", etc to acknowledge participant responses during the meetings.

  4. Response frequency - Set how frequently do you want the AI moderator to use active listening responses.

  5. Acknowledgement responses - Set the exact words to be used for active listening responses.

These settings help maintain a smooth conversation with the participants.

Finalizing Your Settings

Once you have configured all the settings, you can check if these changes are reflected in the AI voice by clicking on "Start Conversation" and talking with AI. Iterate until you're satisfied with your AI configuration.
